# qiankun-admin-example
**qiankun微前端demo，包含基于vue-cli2 和 vue-cli4创建的应用 和 react应用。**

`npm install:all` 安装主子应用所有依赖

`npm start` 启动所有应用

`npm run build` 构建所有应用,最终构建到最外层的dist目录:
